# Multi-Agent AI Research Assistant

## Overview
The Multi-Agent AI Research Assistant is a collaborative and interactive system designed to facilitate research processes by utilizing multiple AI agents. Each agent specializes in a unique aspect of research, enabling efficient data gathering, analysis, and reporting.

## Features
- **Collaboration**: Seamless interaction between different AI agents.
- **Specialization**: Each agent specializes in specific research tasks.
- **Data Analysis**: Automated data collection and analysis capabilities.
- **User-Friendly Interface**: Easy-to-use interface for users to interact with the assistants.
- **Extensible Architecture**: Designed to easily integrate new agents and functionalities.

## Installation
To install the Multi-Agent AI Research Assistant, follow the steps below:
1. Clone the repository:
   ```bash
   git clone https://github.com/jayapriya19-ai/multi-agent-ai-research-assistant.git
   ```
2. Navigate to the project directory:
   ```bash
   cd multi-agent-ai-research-assistant
   ```
3. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the Multi-Agent AI Research Assistant, execute the following command:
```bash
python main.py
```

Follow the instructions in the terminal to interact with the various AI agents.
